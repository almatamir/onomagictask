# Claude Configuration for Guess The Country Project

## About This File
This file configures how Claude AI behaves when working on this project.
When opened in an AI-powered editor like Cursor or Antigravity,
Claude reads this file and follows these rules automatically.
It acts as a persistent memory so I don't need to repeat instructions
in every conversation.

- No unnecessary abstractions - keep files flat and simple

## Project Architecture
The project is split into two separate parts:

### Backend (Python + Flask)
- `app.py` - Flask entry point, registers blueprints
- `routes/game.py` - two endpoints: GET /api/country, POST /api/guess
- `services/database.py` - all MongoDB logic
- `services/gemini.py` - Gemini API, generates countries in bulk
- `seed.py` - one-time script to seed 30 countries into MongoDB
- `data/countries_data.py` - the 30 hardcoded seed countries

### Frontend (Next.js)
- `app/page.tsx` - single page, all UI logic in one file
- Communicates with backend via environment variable NEXT_PUBLIC_API_URL

## Project Stack
- Backend: Python 3.11 + Flask + PyMongo + certifi + google-genai
- Frontend: Next.js + TypeScript + Tailwind CSS
- Database: MongoDB Atlas (free tier)
- AI: Gemini API via google-genai SDK
- Backend runs on port 5000, frontend on port 3000/3001
- Deployed on Railway (backend + frontend)

## Key Technical Decisions
- Gemini is called automatically in the background when countries_left < REFILL_THRESHOLD
- Gemini generates 15 countries in a single API call (not one by one)
- Gemini receives the list of existing countries to avoid duplicates
- Each country gets a `played: true` flag when shown to the user
- count_countries_left() counts only unplayed countries
- If Gemini fails, the existing MongoDB countries are used as fallback
- MongoDB is pre-seeded with 30 countries as a safety net
- New countries generated by Gemini are saved to MongoDB automatically
- SSL certificates fixed using certifi library (required on Mac)
- is_refilling flag prevents multiple simultaneous Gemini calls
- reset_played() is called on every new session (page load) via POST /api/reset
- save_country() always sets played: False on new countries from Gemini
- MAX_COUNTRIES = 300 to prevent unlimited MongoDB growth
